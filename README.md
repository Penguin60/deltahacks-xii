# Delta Control 

An AI augmented 911 dispatcher system to reduce wait times and help more people in need. Check the [Devpost]() for more details.


# Frontend setup

Clone the repo
Have pnpm installed on your system
Cd into frontend
Run pnpm install
Then run pnpm dev and go to 127.0.0.1:3000 to see the UI 


# Backend setup

Cd into backend 
Have uv installed on your system
Run uv venv, then activate the venv
Run uv sync to install packages
Cd to the project root, and run `uvicorn backend.main:app --reload` inside the project root
